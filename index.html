Ascent Stats Image Thingie

Usage:

For ETF2L logs, paste a logs.tf URL in the field

For ESEA stats, visit the state page, right click and save the HTML page, upload it through the file picker

Classes need to be manually set for ESEA, once done, click "Reorder" to sort them

Don't forget to check the score by the way

Click on text to edit it

Click on class icon to cycle it

Click on stats to highlight it

Background size: 1400px*1600px

Logo size: 320px*320px
